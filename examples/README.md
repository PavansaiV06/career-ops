# Job Application Examples
# Career Ops - Healthcare Job Automation Portfolio

## Overview
This repository is a customized version of an AI-driven job application workflow tailored for healthcare and healthcare analytics roles.

## What I Added
- Resume prompt for role-based resume tailoring
- Cover letter prompt for job-specific application writing
- Reusable healthcare skills dataset in `data/skills.md`
- Job-specific examples for multiple target roles

## Target Roles
- Clinical Research Coordinator
- Clinical Data Analyst
- EHR Analyst
- Claims Data Analyst
- SAS Programmer
- System Analyst
- Clinical Research Analyst
- Health Informatics Analyst

## Project Value
This project demonstrates how structured prompts, healthcare domain knowledge, and role-specific examples can be combined to generate tailored resumes and cover letters for different job applications.

## Repository Structure
- `prompts/` → reusable prompts for resume and cover letter generation
- `data/skills.md` → healthcare skills and workflow knowledge base
- `examples/` → sample role-based applications
- `output/` → generated outputs

## Why This Matters
Instead of using the same resume for every application, this workflow helps tailor applications based on job descriptions, healthcare systems knowledge, claims workflows, EHR experience, and analytics skills.
This folder contains tailored resume and cover letter outputs for multiple healthcare and analytics roles.
