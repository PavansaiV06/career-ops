# Job Application Examples

This folder contains tailored resume and cover letter outputs for multiple healthcare and analytics roles.

## Roles Covered
- Clinical Data Analyst
- EHR Analyst
- Claims Data Analyst
- Clinical Research Coordinator
- SAS Programmer

- ## Project Customization

- Built prompt-driven system for resume and cover letter generation
- Created reusable healthcare skills dataset (data/skills.md)
- Developed multiple job-specific application examples
- Focused on roles: Clinical Data Analyst, EHR Analyst, Claims Analyst, SAS Programmer, Clinical Research

## Outcome

This project demonstrates how AI prompts and domain-specific knowledge can be combined to generate tailored job applications for healthcare and analytics roles.

## Purpose
Each example demonstrates how structured prompts and domain-specific skills were used to tailor applications based on job requirements.
